Welcome.

This is a documentation webpage dedicated to the Verse language project. You can find my current progress on the project at Github.

Here you'll find definitions and inner workings of .vrc code, the verse compiler, and its standard library. Intermediate knowledge of the C language and FASM Assembly Language is strongly recommended to understand the compiler code and the assembly output of .vrc files. Click one of the links on the sidebar (if viewing on desktop/tablet) or just below this text to get started.

Compiler Input

Understanding the compiler's behavior allows you to utilize as much of the compiler features later on. If you need help on learning all the compiler flags, or simply compiling code, then this is the right place for you.

Do note that you could simply type the command verse -h or verse --help in the bash shell or command line if you already have the compiler installed.
